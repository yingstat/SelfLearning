Shell&R&Statistics&Bayesian

We are a small self learing group in UDE about essential tools and knowledge in bioinformatics. The reference books we are going to use are as follows: The book for R learning: The Art of R Programming. The book for Shell learning: The Linux Command Line-A complete Introduction. The book for Statistics learning: The Elements of Statistical Learning. The book for Bayesian analysis learning: Doing Bayesian Data Analysis using R and BUGS.
